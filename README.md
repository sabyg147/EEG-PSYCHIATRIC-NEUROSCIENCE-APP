## 🧠 Overview

NeuroScan is an AI-driven clinical system that analyzes EEG signals to detect psychiatric disorders using a hybrid machine learning approach.

It combines:
- ⚡ XGBoost (data-driven predictions)
- 🧬 EEG Biomarker Analysis (domain knowledge)

to deliver fast, interpretable, and accurate results.

---

## 🚀 Key Features

- 🧠 EEG-based psychiatric prediction
- 🤖 Dual-model system (XGBoost + Biomarker)
- ⚡ Real-time inference via FastAPI
- 🌐 Modern clinical-style web interface
- 📊 Interactive dashboard UI
- 📁 CSV upload + manual biomarker input
